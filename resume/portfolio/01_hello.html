Project Hello Quad

3D Graphics

Project Overview

OpenGL과 GLSL을 사용하여 기본적인 함수를 구현하고 Quad를 쉐이더로 구현했다. 또한 fragment shader와 vertex shader를 실시간으로 변경시키는 애니메이션 효과를 적용하였다.

Tasks Completed

  • IndexBuffer, Texture, VertexArray, VertexBuffer 클래스를 완성시켰다.
  • 쉐이더를 사용하여 quad를 만들고 애니메이션 효과를 주었다.
  • C++, GLSL 스킬이 요구되며 애니메이션 효과를 위해 time을 Environment class에서 받아와 sin 과 곱하여 애니메이션 효과를 냈습니다.

Reflection

쉐이더의 기초를 배웠습니다. vertex shader와 fragment shader의 사용법을 익히고 응용했으며 AddVertexBuffer 함수를 구현하는 것이 복잡하여 어려웠습니다. 단순히 의사코드를 따라가기보다 각 함수가 어떤 역할을 하고 다음엔 어떤 함수가 필요할지 생각해보며 구현하니 금방 해결되었습니다.
