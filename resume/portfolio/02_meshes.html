Project Procedural Geometric Modeling

3D Graphics

Project Overview

This assignment implements procedural mesh generation in C++ to render multiple 3D shapes using OpenGL. The core mesh building logic was written from scratch in Mesh.cpp, generating vertex and index data for a plane, cube, sphere, torus, cylinder, and cone. Each shape supports three rendering modes: textured, wireframe, and normal visualization, all driven by a shared index buffer pattern.

Tasks Completed

  • Implemented create_plane_vertices() to generate a grid of vertices with correct position, normal, and UV coordinates by iterating over stacks and slices.
  • Implemented build_index_buffer() to produce triangle indices by splitting each quad into two triangles using a stride based pointer system.
  • Implemented convert_to_lines_pattern() to convert triangle indices into line indices for wireframe rendering.
  • Implemented all six 3D shapes -> Plane, Cube, Sphere, Torus, Cylinder, and Cone. Implemented add_cap() to handle the top and bottom caps of the cylinder and cone.

Reflection

This assignment taught me how 3D geometry is built procedurally from mathematical patterns. The most challenging part was convert_to_lines_pattern() where understanding how a group of 6 indices maps to 4 edges for wireframe rendering required careful thought. Rather than blindly following the pattern, working through each pointer index manually made the logic click. Seeing the same index buffer reused across the sphere, torus, cylinder, and cone also reinforced how powerful a single well-designed function can be.
